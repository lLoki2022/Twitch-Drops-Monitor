# Twitch-Drops-Monitor
**Twitch Drops Monitor** это браузерное расширение для автоматического отслеживания и сбора Twitch Drops и Монет.  Авто-фарм дропов, переключение стримеров с активными кампаниями, уведомления в браузере и Telegram, история полученных наград.  Идеально для экономии времени и не пропускающих награды пользователей Twitch.
